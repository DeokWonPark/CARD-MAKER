### Purpose of the project

- React Hook을 학습하기 위한 실전 프로젝트
- Firebase를 통한 인증 및 데이터베이스 활용방법 학습
- React Router를 사용하여 SPA에서 라우팅 학습
- Cloudinary Api 활용 연습
